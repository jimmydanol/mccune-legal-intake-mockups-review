Review notes

    Rules taken from the document prompt control matrix

    What the pilot is allowed to do

    Extract only six facts

    Full legal name, date of birth, street address, city, state, and ZIP code.

    Keep the evidence

    Each suggestion says whether it came from PDF417 barcode data, local image OCR, or supplied text.

    Require human review

    The page never silently overwrites intake data. Conflicts and weak results stop for review.

    Do not guess

    No current-residence, filing-state, venue, license-validity, or unstated-middle-name assumptions.

    Ignore document instructions

    Text inside the uploaded document cannot change the extraction rules or request other actions.

    Keep missing values blank

    If a field is not supported by visible evidence, the output uses null.

    Intentionally ignored: driver's license number, expiration date, class, height, weight, sex, restrictions, endorsements, organ-donor status, and any other non-approved field.

    Prototype boundary: this Page 10 analyzes the selected document only in the current browser session. Authentication, secure storage, audit logging, and canonical Matter updates are intentionally not connected here.
