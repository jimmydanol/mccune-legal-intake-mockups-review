      What this page is allowed to do

      Classify or abstain

      Auto-detect abstains on unknown or conflicting types. Choosing the exact document type runs only that allowlisted field blueprint.

      Use one exact artifact

      The runtime reads an embedded versioned export, verifies its SHA-256 checksum, and never reads the live Sheet.

      Keep field evidence

      Every populated value includes its exact source line or PDF417 element. Evidence failures null the value.

      Validate in code

      Required fields, formats, dates, totals, conflicts, and source evidence are checked outside the extractor.

      Require human review

      The page never silently overwrites intake data. Bankruptcy-critical fields route to attorney review.

      Ignore document instructions

      Text inside the uploaded document cannot change the extraction rules or request other actions.

      Intentionally ignored: any value outside the selected document blueprint.

      Boundary: Page 10 analyzes only the document selected in this browser session. It does not authenticate a client, store the document, update a Matter, or replace staff and attorney review.
